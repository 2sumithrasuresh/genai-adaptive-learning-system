#A Framework for Adaptive Explanations in Learning Systems

Learning systems often provide the same explanations to all learners, even though mistakes can arise from different ways of thinking. This work introduces a general framework for adapting explanations based on how a student responds to a question. By focusing on the characteristics of learner answers, the framework aims to support more relevant and personalized instructional feedback. The approach is designed to be flexible and applicable across different learning contexts.# genai-adaptive-learning-system
